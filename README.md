This repo houses the Python code to operationalize some of Scott Jarvis's subconstructions of lexical diversity that he and his co-authors have proposed in various publications, starting with (at least):
Jarvis, Scott. 2013. Defining and measuring lexical diversity. In Jarvis, Scott & Daller, M. (eds.), Vocabulary Knowledge: Human ratings and automated measures (Studies in Bilingualism 47), 13–44. Amsterdam: John Benjamins Publishing Company. (https://doi.org/10.1075/sibil.47.03ch1)
Jarvis, Scott. 2013. Capturing the Diversity in Lexical Diversity. Language Learning 63. 87–106. (doi:10.1111/j.1467-9922.2012.00739.x)
Jarvis, Scott & Daller, Michael (eds.). 2013. Vocabulary Knowledge. Amsterdam: John Benjamins Publishing Company.
